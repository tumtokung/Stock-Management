
# Stock System Application




## Tech Stack

 - [React Native](https://reactnative.dev/)
 - [Native Base](https://nativebase.io/)
 - [Firebase](https://rnfirebase.io/)


## Running Project

Support For Android Only ,To run tests, run the following command

```bash
  yarn android
```
